Почему разработчики выбирают torio?

Создан для тех, кто привык ценить быстродействие, автоматизировать задачи и управлять всем с клавиатуры без использования мыши.

Параллельный мультипоиск

Поиск выполняется одновременно по нескольким распределенным базам данных. Если один из источников временно офлайн, система бесшовно продолжит искать на остальных.

Загрузка в фоне и очередь

Задачи выполняются в фоне и не блокируют поисковую сессию. Вы можете продолжать серфинг по категориям и топам, пока файлы загружаются в целевую директорию.

Авто-раздача по правилам

Полная поддержка стандартов BitTorrent-сетей. После успешного скачивания утилита переходит в режим сидирования, помогая сообществу, но функцию легко отключить одним нажатием.

Запуск в три простых шага

Не требуется ставить сторонние тяжелые GUI-клиенты и раздувать операционную систему. Вам нужен только установленный Node.js v22+.

1

Убедитесь в наличии среды

Версия Node.js должна быть не ниже 22-й. Проверьте через команду node -v.

2

Разовый быстрый запуск

Запустите интерактивную сессию в один клик без глобального скачивания пакета через NPX.

3

Глобальная установка

Зарегистрируйте псевдоним в системе команд и вызывайте из любого каталога через вызов torio.

terminal ~ setup.sh
bash/zsh
# 1. Проверить версию Node.js
$ node -v
v22.2.0
# 2. Быстрый локальный старт
$ npx torio-cli
Инициализация реестра... запуск...
# 3. Глобальная установка и запуск по короткой команде
$ npm install -g torio-cli
✔ Added 1 package in 0.94s
$ torio
